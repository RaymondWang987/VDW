Download For Metadata

We present VDW metadata (excluding processed movies and annotations) for downloading. Since VDW is quite large (over 2 million frames and 8TB on hard drive), we only release the VDW Testing Set in the first stage. The VDW Training Set will be gradually prepared, organized, and released movie by movie (or animations, documentaries, web videos).

For the metadata of each sequence, we provide IMDB numbers, starting time, end time, movie durations, resolutions, cropping area, etc. We will also build a GitHub repo (Coming Soon) to release the VDW toolkits (including Video Preprocessing, disparity annotations, etc.). In this way, you can restore some sequences of VDW or build your own data from stereo videos.

Application For VDW Dataset

The release of the VDW dataset is under strict conditions. Our VDW dataset cannot be used or distributed for any commercial purposes. Please refer to our License.

If you want to use our processed VDW dataset for your own research. We provide an application form template in Application. You should fill in the application form and send the application to VDW's official email: vdw.dataset@gmail.com. Your name, institution, purpose for using our data, and agreement to our License will be included in the application template. You should also sign the application form with your electronic signature. We will examine your application and send you feedback on 3-5 weekdays.

At this stage, our VDW Training Set is not ready, because organizing and uploading 8TB training data are extremely heavy and time-consuming. For the applications in this stage, if you need our training data and your application is approved by us, we will register your information and send you the required data when we are ready.
