Download and Applications

We plan to release the VDW dataset under strict conditions. VDW dataset is licensed under CC BY-NC-SA 4.0, which can only be used for academic and research purposes. It cannot be used for any commercial purposes. Since our VDW dataset is quite large (2.23 million frames, over 8TB on hard drive), we will gradually organize and release our VDW dataset for the community.

We first release the VDW test set with 90 videos and 12622 frames. We will gradually organize and upload the VDW training set organized by data source (each movie, documentary, animation, or web video).

The release of VDW dataset will be divided into three parts: (1) Downloading for metadata; (2) Application for our processed VDW Dataset; and (3) Toolkits for data usage or constructions.

Download For Metadata

We present VDW metadata for downloading. Downloading our metadata represents your agreement to our license and conditions. For the metadata of each sequence, we provide movie name, IMDB numbers, resolutions, movie durations (second), sequence start time, end time, cropping area (Four coordinate values from the upper left corner to the lower right corner). The downloading of metadata is presented as follows:

  1. VDW Test Set
  2. VDW Training Set (Gradually updating, Coming soon).

Application For VDW Dataset

If you want to use our processed VDW dataset for your own research. We provide an application form template. You should fill in the application form and send the application to VDW's official email: vdw.dataset@gmail.com. We will examine your application and send you feedback on 3-5 weekdays.

At this stage, our VDW training set is not ready, because the work for organizing and uploading 8TB training data are extremely heavy. For the applicants in this stage, if you need our training data and your application is approved by us, we will register your information and send you the required data when we are ready.
